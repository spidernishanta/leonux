# Leonux Website

A modern, professional freelancing website for Leonux - transforming ideas into digital success.

## Features

- **Multi-page website** with Home, Services, About Us, and Contact pages
- **Dark theme** with cyan blue accents (#00d4ff)
- **Fully responsive** design for all devices
- **Modern UI** with glassmorphism effects and smooth animations
- **Interactive elements** including hover effects and transitions
- **WhatsApp integration** for easy contact

## Pages

### Home Page (index.html)
- Hero section with compelling call-to-action
- Our Expertise showcase (Website Development, Design, Automation)
- Why Choose Leonux features
- Recent Work portfolio
- Client testimonials
- CTA section

### Services Page (services.html)
- Comprehensive service listings
- Detailed descriptions for each service
- "How It Works" process flow with circular icons
- All services include: Custom Web Development, E-Commerce, CMS, Responsive Design, Maintenance, SEO, UI/UX, Web Apps, API Integration

### About Us Page (about.html)
- Company introduction and mission
- Our Story section
- Core Values (Client-Centric, Quality, Transparency, Sustainability)
- Project showcase
- Why Work With Us section

### Contact Page (contact.html)
- Contact introduction
- WhatsApp quick contact button
- Direct contact methods (Email, Phone, Instagram)
- Decorative design elements

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Modern styling with:
  - CSS Grid and Flexbox for layouts
  - CSS Variables for theming
  - Gradients and backdrop filters
  - Smooth transitions and animations
  - Responsive design with media queries

## How to Use

1. Open `index.html` in your web browser to view the homepage
2. Navigate between pages using the header navigation menu
3. All pages are fully functional and ready to deploy
